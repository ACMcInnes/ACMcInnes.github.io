Project - Holiday Website Mockup

Brief - Design website using Sketch to them code using HTML and CSS

Header Banner

Header banner and featured destinations for easy access. Large inviting feature image gets customers thinking about their next holiday. Large buttons and text provide obvious suggestions and an action for the customer as they first land on the page.

Purchase Packages

Cruise packages - quickly convert prospects into customers with easy all-in-one travel packages. Layout includes a hero image to draw the eye and shows all the information they need so there is no sneaky suprises later in the checkout process.

Search and Filters

Accomodation - Let customers filter to find the perfect spot to compliment their holiday. Easy to use filters, display most relevant options first, then allow customers to scroll through additional options.

Upsell Packages

Adventure Packages - Upsell extras to customers to make their holiday planning easier and to drive home a one-stop-shop for all your holiday needs.
